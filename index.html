💵📊Cost clasifier

Дипломний проект. Сервіс, який проводить аналіз і класифікує внесені витрати. Проводить порівняння кількості витрат по проміжуткам часу, що допомогає зручно оцінити економічну ситуацію і продумати інтенсивність наступних витрат.

Однією з головних функцій є можливість інтергації з такими банками як: Monobank, що дозволяє автоматиузати процес внесення витрат

Проект розробляється по методології TDD (Test-Drive Development), що забазпечує майже повне покриття тестами. Під час розробки використовуються також такі технології як CI/CD Github Actions, та інші допоміжні фреймворки та бібліотеки, які тестують, форматують та перевіряють код. Середних них є такі технології як: pytest, flake8, pylint, black, isort, bandit. Всі вони автоматично викликаються під час роботи з git за допомогою pre-commit, що робить розробку зручною та ефективною.
